Adding Games in Bulk

  • Add your games in the text field.
  • Games should be either separated by commas or copy/pasted from a spreadsheet
  • Game names will be cross-referenced with their official titles on Board Game Geek. The more precise your game names are, the better!
  • If a game cannot be found, you will need to manually input it.
